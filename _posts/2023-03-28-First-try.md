## Data Preparing

```bash
export analysisdir=`pwd`
export SAS_ODF=$analysisdir

export DYLD_LIBRARY_PATH=$SAS_DIR/lib:$SAS_DIR/libextra
cifbuild
export SAS_CCF="$analysisdir/ccf.cif"
odfingest
export SAS_ODF="$analysisdir/`ls *.SAS`"
```
